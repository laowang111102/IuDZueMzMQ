# 简易考勤系统 python708

一款基于 Python 开发的学生考勤管理系统，具备用户注册、登录以及考勤信息管理等功能。

## 技术栈

- Python

## 功能模块

- 用户注册
- 用户登录
- 考勤信息管理
  - 录入
  - 查询
  - 修改
  - 删除
  - 批量查看

## 系统角色

- 学生用户

## 运行环境

- Python 环境搭建
  - Python 3.x

## 部署步骤
